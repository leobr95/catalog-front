# Documentación Front

## Stack
- Next.js (App Router)
- Ant Design
- Redux Toolkit
- SweetAlert2

## Features (Prueba técnica)
- Login demo: `/api/auth/token`
- CRUD categorías: `/api/categorias`
- CRUD productos + paginación + filtros: `/api/productos`
- Bulk import: `/api/productos/masivo`
- MegaMenu (categorías + búsqueda)
- Modales para crear/editar
